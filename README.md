# Компьютерная графика 2

## Запуск
1. Скачать статическую библиотеку для вашей ОС и поместить в `third_party/lib`
```
https://github.com/raysan5/raylib
```
2. Собрать проект
```
cmake -B build && cmake --build build
```
3. Запустить проект
```
make -C build && build/computer-graphics-4
```
